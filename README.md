# 仓库管理系统 java3

## 项目概述

仓库管理系统是一款基于Web的企业级应用，采用前后端分离的技术架构，主要服务于企业内部物品的出入库、库存管理以及相关角色的权限管理。

## 技术栈

- **架构**: B/S、MVC
- **开发语言**: Java
- **后端框架**: SpringBoot
- **前端框架**: Vue
- **数据库**: Mysql
- **数据库操作**: Mybatis-Plus

## 功能模块

### 系统角色

- 用户
- 管理员
- 超级管理员

### 功能列表

- 登录功能
- 物品出入库功能
- 用户管理功能
- 管理员管理功能
- 仓库管理功能
- 分类管理功能
- 库存预警功能
- 仓库数据功能
- 物品数据功能

## 系统环境与开发工具

- **系统环境**: Windows
- **开发环境**: 
  - IDEA
  - JDK1.8
  - Maven3.6.1
  - Mysql5.7/8.0
  - navicat17
  - node18
